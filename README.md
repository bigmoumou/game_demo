# 戰國妖狐傳 RPG Demo

GitHub Pages-ready static RPG MVP.

## Controls

- `WASD` or arrow keys: move
- `Space`: interact / advance dialog
- `J` or `Attack`: attack

## Demo Flow

1. Talk to the merchant in the forest.
2. Go left to the shrine and collect the foxfire charm.
3. Go right to the battlefield and defeat the three-phase boss.
4. Return to the forest for the ending.
