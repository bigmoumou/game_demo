# 執行規範

本專案在進行 AI 圖片或素材生成時，必須嚴格遵守以下執行規範：

1. **背景生成規範**<br>
   當使用者要求生成「背景」時，預設使用 `--type background` 參數，並指定解析度為 `800x600`。

2. **角色與怪物生成規範**<br>
   當使用者要求生成「角色/怪物」時，預設使用 `--type spritesheet` 參數。

3. **模型與風格統一**<br>
   必須強制加入參數 `--model "nano banana 2"`，以確保專案中的像素風格（Pixel Art）完全統一。

4. **Sprite Sheet 動畫規格**<br>
   生成的 Sprite Sheet 必須包含四個方向（下、左、右、上）的行走動畫（Walk Cycle），且每個方向需包含 `3-4 幀`。

5. **儲存路徑與去背要求**<br>
   所有生成的圖片請自動儲存至 `assets/sprites/` 目錄，並確保角色背景完成去背（透明背景，transparent）。
